Basics

Load Core

Load your core by clicking on the first tab. Scroll down until you reach the desired Core. Here's a list of some of the available cores:

  • NES: Nestopia
  • Game Boy / Color: Gambatte
  • Super Nintendo: Snes9x
  • Sega Genesis: Genesis Plus GX
  • PlayStation: Beetle PSX
  • Nintendo 64: Mupen64 Plus
  • 2048: 2048 Puzzle Game
  • Arduous: Arduboy
  • BK: Elektronika BK
  • BlueMSX: MSX/MSX2
  • ChaiLove: ChaiLove Game Framework
  • Craft: Minecraft-like game for libretro
  • DeSmuME: Nintendo DS
  • DOSBox: MS-DOS
  • EasyRPG: RPG Maker 2000/2003
  • ECWolf: Wolfenstein 3D
  • FB Alpha 2012: Arcade games (various)
  • FB Alpha 2012 CPS1: Capcom Play System 1
  • FB Alpha 2012 CPS2: Capcom Play System 2
  • FB Alpha 2012 NeoGeo: Neo Geo
  • FCEUmm: Nintendo Entertainment System
  • FFmpeg: Multimedia (Video, Audio)
  • FreeChaF: Fairchild Channel F
  • Game Music Emu: Game music player
  • GearBoy: Game Boy / Game Boy Color
  • GearColeco: ColecoVision
  • GearSystem: Sega Master System / Game Gear
  • Genesis Plus GX: Sega Genesis / Mega Drive
  • Genesis Plus GX Wide: Sega Genesis / Mega Drive (wide aspect ratio)
  • GLupeN64: Nintendo 64
  • Handy: Atari Lynx
  • JAXE: JAXE Engine
  • Jump 'n Bump: Jump 'n Bump game
  • LowResNX: LowRes NX Game Creator
  • Lutro: Lua game framework
  • MAME 2000: Arcade games (various)
  • MAME 2003: Arcade games (various)

Remember that the content must be compatible with the matched Core.

Load Content

After selecting Core, click Run. After RetroArch opens, click Add Content and select your compatible ROM.

  • Nestopia > YourGame.nes
  • Gambatte > YourGame.gbc
  • Snes9x > YourGame.smc
  • Genesis Plus GX > YourGame.md
  • Beetle PSX > YourGame.cue
  • Mupen64 Plus > YourGame.z64
  • etc.

Cleanup Storage

The trashcan erases your existing configuration and presets. If the Web Player doesn't start, you should click the trashcan and refresh the cache in your browser (usually F5 or Shift+F5).

Quick Menu

If you click on the three line icons, the Quick Menu will open here as in RetroArch.
